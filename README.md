Built using:
- React
- NodeJs
  
In the /Document-Management-System/client, run:
### `npm install`
### `npm start`

In the /Document-Management-System/server, run:
### `nodemon server.js`

Run the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
